# Skill: Daily Report

## Purpose
Produce an end-of-day summary of team activity for the human operator.

## Process

1. Review all agents' standup entries for the day
2. Compile into a single summary
3. Highlight: completed items, in-progress work, blockers, decisions made
4. Note anything requiring human attention

## Format

```
# Daily Report — [Date]

## Completed Today
- [item 1]
- [item 2]

## In Progress
- [item 1] — [status/ETA]

## Needs Your Attention
- [item requiring human input]

## Decisions Made
- [decision] — [reasoning]

## Tomorrow
- [planned priorities]
```
